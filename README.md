[ブログ記事](http://inside.pixiv.net/entry/2016/07/05/194025)

## インストール

```sh
git clone https://github.com/hakatashi/slack-ikku.git
cd slack-ikku
npm install
cp config.example.json.ls config.json.ls
vi config.json.ls
```

## 起動

```sh
npm start
```
